Tugas Modul 5 - Bootstrap

Implementasi Grid, Table, Image, dan Button menggunakan Bootstrap 5.3.0

1. Instalasi Bootstrap (CDN)

Berhasil Terpasang! Berkas ini menggunakan Bootstrap v5.3.0 (CSS & JS Bundle) dan jQuery v3.7.0 melalui CDN resmi.

Pustaka CSS dimuat di dalam tag <head>, sementara jQuery dan Bootstrap JavaScript dimuat di bagian akhir berkas <body>.
