Tugas 4 - jQuery AJAX

Implementasi Metode .load() dan $.ajax() menggunakan jQuery dan Web Server

Catatan Keamanan (CORS): Halaman ini dijalankan melalui server web lokal (seperti Python HTTP Server atau Apache XAMPP). AJAX tidak akan berjalan jika Anda membuka berkas HTML secara langsung dari folder komputer Anda (menggunakan protokol file:///) karena diblokir oleh kebijakan keamanan browser modern.

1. Metode jQuery .load()

Metode AJAX sederhana untuk meminta data dari server dan memasukkan konten HTML yang dikembalikan langsung ke elemen target.

Konten ajax_info.txt belum dimuat. Klik tombol di bawah...

$(document).ready(function() {
  $("#changeBtnLoad").click(function() {
    $("#demo-load").load("ajax_info.txt");
  });
});

2. Metode jQuery $.ajax()

Fungsi AJAX lengkap yang menawarkan kontrol mendalam atas parameter request (GET/POST, header, data, timeout, sukses/gagal callback).

Konten ajax_info.txt belum dimuat. Klik tombol di bawah...

$.ajax({
  url: "ajax_info.txt",
  type: "GET",
  dataType: "html",
  success: function(result) {
    $("#demo-ajax").html(result);
  },
  error: function(xhr, status, error) {
    $("#demo-ajax").html("Error: " + error);
  }
});
