Tugas 3 - Pustaka jQuery

Implementasi jQuery CDN, Efek Hide/Show, dan Animasi Tinggi (Height Toggle)

1 & 2. Cara Kerja jQuery (Hello World)

Menggunakan event $(document).ready() untuk mengeksekusi fungsi setelah struktur dokumen HTML sepenuhnya dimuat.

Klik tombol di bawah untuk memicu respon jQuery...

$(document).ready(function() {
  $("#hello-output").html("Hello, World!");
});

6.6.1. Efek Hide / Show

Menggunakan selektor ID untuk menangkap klik tombol, kemudian memanggil efek hide() dan show() pada tag paragraf.

Jika Anda mengklik tombol "Hide", teks paragraf ini akan menghilang.

$("#hide").click(function() {
  $("p").hide();
});
$("#show").click(function() {
  $("p").show();
});

6.6.2. Efek Animasi (Toggle Height)

Menggunakan metode animate() dengan parameter { height: 'toggle' } untuk menyembunyikan dan memunculkan kotak elemen dengan efek meluncur.

Klik tombol berulang kali untuk melihat transisi tinggi kotak di bawah ini.

Secara default, elemen HTML berposisi statis. Untuk memanipulasi posisinya secara absolut, atur position: absolute pada elemen tersebut.

$("button").click(function() {
  $("div").animate({
    height: 'toggle'
  });
});
